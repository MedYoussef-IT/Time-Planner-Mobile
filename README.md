TimePlanner is a productivity Android application designed to help users plan one-time or recurring events.

While the core scheduling features were implemented, the project was paused due to unstable notification behavior related to Android background processes. The project remains a valuable learning experience.

## Features
- One-time and recurring event planning
- Basic scheduling system
- Local notifications (unstable)

## Technologies Used
- Java
- Android SDK

## Project Status
Not Finished

## Purpose
Built to learn event scheduling and notification handling in Android, highlighting real-world development challenges.
